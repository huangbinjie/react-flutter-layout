## Stack

提供可以重叠的布局效果。子组件有 none-positioned 的组件时，高宽是 none-positioned 子组件高宽。
子组件没有 none-positioned 时，宽度会占满父容器。

## Usage

<Playground>
  <Container width="200px" height="200px" color="#ececec">
    <Stack>
      <Positioned top="0px" left="0px">左上</Positioned>
      <Positioned top="0px" right="0px">右上</Positioned>
    </Stack>
  </Container>
</Playground>

### 有 none-positioned 组件时

<Playground>
  <Container width="200px" height="200px" color="#ececec">
    <Stack>
      <Container width="100px" height="100px">none-positioned</Container>
      <Positioned top="0px" left="0px">左上</Positioned>
      <Positioned top="0px" right="0px">右上</Positioned>
    </Stack>
  </Container>
</Playground>
